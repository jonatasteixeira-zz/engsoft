2.1 - ELICITAÇÃO DE REQUISITOS

Elicitação de requisitos é uma fase do projeto onde são extraídas informações do cliente sobre o que ele deseja que seja construído.

2.1.1 - Elicitação, validação e gerenciamento de requisitos.

Técnicas de elicitação de Requisitos

  • VORD (Viewpoint-oriented requirements definition)
  • Etnografia
  • Workshops
  • Prototipagem
  • Entrevistas
  • Questionários
  • Brainstorming
  • JAD (Joint Application Design)
    • Dinâmica de grupo
    • Manutenção do processo organizado e racional
    • Utilização de documentação padrão

2.1.1 - Elicitação, validação e gerenciamento de requisitos.

Validação de Requisitos

  • O não seguimento das normas de qualidade do projecto e da empresa
  • Descrição pouco clara dos requisitos
  • Falhas na modelação dos requisitos
  • Conflitos entre requisitos que não foram detectados no processo de análise
  • Requisitos não realistas
  • Falta de informação

Gerenciamento de Requisitos

O gerenciamento de requisitos é um modelo sistemático para encontrar, documentar, organizar e rastrear os requisitos variáveis de um sistema.
